# 피처폰 게임 런처

피처폰 게임을 모아서 간단하게 설치하고 실행할 수 있도록 도와주는 PC용 런처입니다.

이 도구는 안드로이드 에뮬레이터를 이용해, 고전 피처폰 게임을 쉽게 설치하고 실행할 수 있게 해 줍니다.

> ⚠️ 이 프로젝트는 개인적인 추억·보존 용도를 위한 것입니다.  
> 저작권자 또는 권리자 측에서 문제 제기나 삭제 요청이 있을 경우,  
> 관련 파일 및 저장소는 지체 없이 삭제하거나 수정할 예정입니다.

---

## 개요

- 여러 피처폰 게임을 한 곳에 모아 관리
- 에뮬레이터에 APK를 간단한 절차로 설치
- 원하는 게임을 클릭해서 실행할 수 있는 간단한 GUI 제공
- 개인적인 향수, 보존 목적에 맞춰 제작

# Feature Phone Game Launcher

A simple installer and launcher for feature phone games on PC.

This tool helps you install and launch classic feature phone games easily using an Android emulator.

> ⚠️ This project is intended for personal, nostalgic use only.  
> If any copyright holder or rightsholder raises an issue,  
> the related files and repository will be taken down or modified without hesitation.

---

## Overview

- Collects multiple feature phone games in one place
- Installs APKs to the emulator with minimal steps
- Provides a simple GUI to select and launch games
- Designed for personal nostalgia and preservation use
